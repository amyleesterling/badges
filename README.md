# Pyr Badge Editor

Standalone shareable repo for the Pyr badge editing system.

Live site:

- https://amyleesterling.github.io/badges/

What it does:

- Browse all Pyr badges in one editor.
- Edit descriptions and image prompts inline.
- Download updated `manifest.json` and `center-image-prompts.csv`.
- In Chromium-based browsers, save directly back into a local badge folder with the folder picker.

Notes:

- Firefox and Safari can use the download buttons if direct folder saving is unavailable.
- This repo contains the static editor and badge assets only. It does not include any API keys.
